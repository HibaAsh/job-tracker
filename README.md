# Job Tracker - Full Stack Application

A complete job application tracking system with React frontend and JSON Server backend. Track your job search with CRUD operations, search, filters, pagination, and dark mode.

## Live Demo

🔗 [View Live App](https://job-tracker-rupt.vercel.app/)

## Repository Structure

job-tracker/
├── frontend/ # React application
├── backend/ # JSON Server API
└── README.md # This file

## Features

- ✅ Create, Read, Update, Delete job applications
- 🔍 Search by company or role
- 🏷️ Filter by status (Applied, Interview, Offer, Rejected)
- 📄 Pagination (5 jobs per page)
- 🌓 Dark/Light mode with persistence
- 📊 Real-time statistics
- 📱 Fully responsive design
- 🚀 Deployed on Vercel (frontend) & Render (backend)

## Tech Stack

| Part | Technology |
|------|------------|
| Frontend | React 19, Vite, React Router DOM, Axios |
| Backend | JSON Server |
| Styling | Inline styles with theme context |
| Deployment | Vercel (frontend), Render (backend) |
